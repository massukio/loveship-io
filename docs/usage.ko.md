# LoveShip.io — 사용 가이드

> 코드 리뷰는 칼같이 하면서 "아무거나 먹자"를 진짜 아무거나로 해석했다.

## LoveShip이란?

LoveShip은 AI 연애 어드바이저입니다. 파트너를 더 잘 이해하고, 잊을 수 없는 데이트를 계획하고, 중요한 세부 사항을 기억하고, 더 효과적으로 소통할 수 있도록 도와줍니다. AI가 시간이 지남에 따라 당신의 관계를 학습합니다.

## 시작하기

### 1. 설정 (처음 한 번만)

AI에게 말하세요:

> "LoveShip 설정해줘"

질문 내용:
- **언어**: 선호하는 언어 선택 (한 번만 묻고 영구 저장)
- **데이터 디렉토리**: 데이터 저장 위치 (기본값: `~/.loveship/`)
- **내 프로필**: 이름, 성별, 나이, 직업
- **상대방 프로필**: 동일 (어프로치 중이면 건너뛰기 가능)
- **관계 상태**: 어프로치 중, 교제 중, 기혼, 육아 중
- **선호도**: 식이 제한, 사랑의 언어 (각각 상위 3개), 금기 영역 (피해야 할 주제)

나중에 수정하려면: "LoveShip 프로필 수정해줘"라고 말하면 됩니다

### 2. 자연스럽게 말하기

명령어가 필요 없습니다. LoveShip에게 필요한 것을 말하면 됩니다:

#### 메시지 해독

파트너가 혼란스러운 메시지를 보냈을 때:

> "그녀가 '괜찮아'라고 했는데, 무슨 뜻이야?"
> "'알아서 해'라고 답장이 왔어. 어떻게 답해야 해?"
> "'아무거나'라고 했는데 진짜 아무거나야?"

결과:
- **문자 그대로의 의미** — 단어 자체의 뜻
- **감정적 의도** — 실제로 느끼는 것
- **추천 답변** — 파트너의 사랑의 언어로 순위가 매겨진 2-3개 옵션

#### 데이트 장소 찾기

> "서울에서 데이트 계획 세워줘"
> "도쿄 레스토랑 찾아줘, 예산 높게, 모험적인 분위기로"
> "부산에서 좋은 데이트 장소?"

파트너의 취향에 맞는 이유가 포함된 엄선된 3곳을 받을 수 있습니다.

#### 아이스브레이커, 선물, 대화 주제

> "등산 좋아하는 사람에게 줄 선물?"
> "수의사로 일하고 주말에 동물 보호소에서 봉사활동——어떻게 인상 남길 수 있어?"
> "두 번째 데이트 대화 주제"

결과:
- 어색하지 않은 3개의 아이스브레이커
- 가격대별 3개의 선물 아이디어
- 깊이 있는 3개의 대화 주제

**참고**: LoveShip은 절대로 조작적인 기법을 제안하지 않습니다. PUA, 비하, 심리 게임 없음.

#### 기억 저장 및 검색

**기억 저장:**
> "기억해줘: 그녀는 말차 라떼를 가장 좋아해"
> "그의 생일은 7월 15일이야"
> "교토에 가고 싶다고 했어"

기억은 자동으로 태그가 붙고, 날짜는 자동 감지됩니다.

**기억 검색:**
> "말차에 대해 뭘 저장했지?"
> "생일 기억 떠올려줘"
> "레스토랑으로 저장한 게 뭐야?"

#### 피드백

제안 후, LoveShip에 결과를 알려주세요:

> "그 제안 잘 됐어!" — 긍정적 피드백
> "안 맞았어" — 부정적 피드백
> "그녀가 비건이 됐어" — 정보 수정
> "조용한 곳을 좋아해, 시끄러운 건 싫어해" — 선호도 조정

## Feedback 효과적으로 사용하기

피드백은 LoveShip이 더 똑똑해지는 핵심입니다:

- **긍정적**: 효과적이었던 것을 알려주세요. 비슷한 제안을 더 많이 합니다.
- **부정적**: 맞지 않았던 것을 알려주세요. 무엇이 문제였는지 물어봅니다 (건너뛰기 가능).
- **수정**: 가장 강력한 유형. LoveShip이 프로필 업데이트를 감지하면, 제안된 변경 사항을 보여주고 확인을 요청합니다. **자동 업데이트는 절대 하지 않습니다**.

### 베스트 프랙티스

1. **구체적으로** —— "안 좋아했어"보다 "레스토랑이 너무 시끄럽다고 했어"가 유용
2. **데이트 후 피드백** —— 실제 데이트 결과를 알려주세요
3. **변화를 즉시 업데이트** —— 식이 변화, 새로운 알레르기, 관계 이정표
4. **자연스러운 말로** —— 친구에게 말하듯이 쓰세요

## 쓸수록 정확해져요

프로필은 처음에는 간단한 스케치입니다. LoveShip을 많이 사용할수록:
- 기억이 파트너에 대한 세부 사항을 축적
- 피드백이 선호도를 조정하고 실수를 수정
- 제안이 점점 더 개인화

공유하고 피드백을 줄수록 LoveShip이 당신의 관계를 더 잘 이해합니다.

## 데이터 및 개인정보

- 모든 데이터는 로컬 머신에 저장: `~/.loveship/`
- 서버로 전송되지 않음 (선택한 AI API 제외)
- 언제든지 모두 삭제 가능: `rm -rf ~/.loveship/`
- 기억은 추가 전용 — LoveShip은 항목을 삭제하지 않음

## 팁

- LoveShip은 14일 이내의 생일과 기념일을 자동으로 리마인드
- 작은 것들을 많이 저장 — 쌓이면 제안 품질이 크게 향상
- LoveShip이 프로필과 충돌을 감지하면 일시 중지하고 업데이트 전에 확인
- "LoveShip 프로필 수정해줘"라고 하면 언제든지 설정 업데이트 가능
