> 她傳了「隨便你」，你回了「好」。從此你們三天沒說話。

Your AI relationship advisor. Just talk naturally — LoveShip figures out what you need.

---

### Capabilities

- **Setup** — Interactive profile wizard: language, partner details, love languages, no-go zones
- **Translate** — Decodes "I'm fine" into what they actually mean, with suggested responses
- **Plan Date** — Real venue search tailored to dietary needs, budget, vibe, and relationship stage
- **Impress** — Icebreakers, gift ideas, and deep conversation topics (zero PUA)
- **Remember** — Memory vault with auto-tagging and date detection
- **Feedback** — Learning loop that actually improves future suggestions
- **Context** — Silent co-pilot that loads your profile and surfaces upcoming dates within 14 days

### Highlights

- **5 tools supported** — Claude Code, VS Code, Cursor, Codex, OpenClaw (personal or project scope)
- **Context-first** — Asks before guessing. Missing context = clarifying questions, not vague answers
- **Auto-save memory loop** — Every suggestion and outcome saved automatically. The more you use it, the better it gets
- **Progressive accuracy** — Profile starts as a sketch, gets sharper with every interaction
- **5 languages** — English, 繁體中文, 简体中文, 日本語, 한국어
- **Local data only** — Everything at `~/.loveship/`. Nothing leaves your machine

### Install

```bash
git clone https://github.com/massukio/loveship-io.git
cd loveship-io
bash scripts/install.sh
```

Then tell your AI: **"set up LoveShip"**
